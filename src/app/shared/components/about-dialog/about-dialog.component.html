About

PMvite is a collaborative project management app in MVP development stage. PMvite's Role Based Authorization and the provided access levels let the project owners and admins to safely share the projects and tasks with their team members and contributors.
Tech Stack
MongoDB Atlas, Google Cloud Platform, NodeJs, Express (RESTful API), Auth0, Typescript, Angular and Material design. The following Google Cloud services are utilized for security and scalability purposes: Cloud Run, Cloud Storage, Cloud Secret Manager, Cloud Build.
