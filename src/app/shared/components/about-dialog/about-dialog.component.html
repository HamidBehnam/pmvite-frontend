About

PM is a demonstrative project management app to focus on different levels of software development process in the context of a demonstrative yet fully functional application. From MongoBD and authentication to REST APIs and role based authorizations, from Jenkins and CI/CD to Frontend and UI implementations. Tech stack: MongoDB, NodeJs, Express (RESTful API), Auth0, Typescript, Angular and Material design. Jenkins and Rundeck are utilized for continuous integration and deployment. Application is hosted on Google Cloud (Compute Engine).
Designed and Implemented by Hamid Behnam.
